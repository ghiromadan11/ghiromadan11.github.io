Dashboard Pemantauan Harga Pangan (Early Warning System)

Peran Fullstack Web Developer
Teknologi CodeIgniter 4, Chart.js, AJAX, MySQL
Tahun 2025

Deskripsi Proyek

Sistem ini merupakan Early Warning System berbasis web yang digunakan untuk memantau harga komoditas pangan secara real-time. Dashboard menampilkan grafik interaktif dan riwayat harga berdasarkan wilayah dan tanggal input.

Fitur Utama

  • Input dan update harga komoditas
  • Grafik batang harga berdasarkan riwayat input
  • Status tren harga (Naik / Turun / Stabil)
  • Dashboard admin, Kepala, dan user
  • Popup grafik interaktif menggunakan Chart.js

Tanggung Jawab Saya

  • Merancang struktur database
  • Mengembangkan backend menggunakan CodeIgniter 4
  • Mengimplementasikan AJAX untuk pengambilan data grafik
  • Mengintegrasikan Chart.js untuk visualisasi data
  • Mendesain UI modern dan responsif

Hasil & Dampak

Sistem ini membantu monitoring harga pangan menjadi lebih cepat, akurat, dan informatif. Visualisasi grafik mempermudah analisis tren harga untuk pengambilan keputusan.
